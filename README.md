# Análisis de Datos del Titanic 

¡Hola! Este es el repositorio oficial de nuestro proyecto. Aquí realizamos un análisis completo sobre el famoso naufragio del Titanic, utilizando Python y la librería Pandas para entender qué factores influyeron en la supervivencia de los pasajeros.

## Objetivos del Trabajo
* Aplicar herramientas de programación en Python para analizar bases de datos reales.
* Aprender a trabajar en equipo de forma organizada utilizando Git y GitHub sin romper el código de los compañeros.
* Encontrar patrones interesantes (como si la edad, el sexo o la clase del tiquete afectaron la oportunidad de sobrevivir).

## ¿Cómo ejecutar el código?
Para ver nuestro análisis en tu computadora, sigue estos pasos:
1. Abre este proyecto en tu editor de código favorito (nosotros recomendamos **Visual Studio Code**).
2. Asegúrate de tener instalado Python y la librería **Pandas**.
3. Descarga los archivos de datos (`train.csv` y `test.csv`) siguiendo las instrucciones que dejamos dentro de la carpeta `data/`.
4. Abre el archivo dentro de `notebooks/Proyecto_PRJ.ipynb` y ve dándole al botón de **Play** a cada celda de arriba hacia abajo.

## Estructura del Proyecto
Organizamos nuestro repositorio de la siguiente manera para que sea fácil de revisar:
* **`notebooks/`**: Contiene el cuaderno principal con todo el flujo del análisis y los códigos.
* **`data/`**: Carpeta destinada a los datos (protegida para no subir archivos pesados).
* **`docs/`**: Aquí guardamos los informes, el PDF final y la documentación de los conflictos resueltos.
* **`outputs/`**: Aquí se guardan automáticamente los gráficos y tablas de resultados generados.
